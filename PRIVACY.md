# Política de Privacidade – Loterias Inteligentes

Última atualização: 2026

## 1. Informações Coletadas

O aplicativo **Loterias Inteligentes** não coleta, armazena, compartilha ou transmite dados pessoais dos usuários.

O aplicativo funciona totalmente offline e utiliza apenas dados históricos públicos de sorteios de loterias e arquivos locais necessários para o funcionamento do programa.

Nenhuma informação pessoal do usuário é coletada.

---

## 2. Processamento de Dados

Todo o processamento realizado pelo aplicativo ocorre localmente no dispositivo do usuário.

O aplicativo:

- não envia dados para servidores externos;
- não utiliza serviços de análise (analytics);
- não utiliza rastreamento;
- não exibe publicidade;
- não coleta informações do usuário.

---

## 3. Permissões do Sistema

O aplicativo pode acessar o sistema de arquivos do dispositivo apenas para:

- ler arquivos da base de dados de sorteios;
- salvar configurações do aplicativo;
- permitir que o usuário exporte ou salve jogos gerados.

Essas permissões são utilizadas exclusivamente para o funcionamento do aplicativo.

---

## 4. Privacidade do Usuário

O uso do aplicativo é totalmente anônimo.

Não é necessário:

- criar conta;
- fornecer e-mail;
- informar telefone;
- inserir qualquer dado pessoal.

---

## 5. Uso do Aplicativo

O **Loterias Inteligentes** é um software de análise estatística e geração de combinações de números para loterias.

O aplicativo não garante prêmios ou ganhos em jogos de loteria.

---

## 6. Isenção de Responsabilidade

O software é fornecido **"no estado em que se encontra" (as is)**, sem garantias de qualquer tipo.

O desenvolvedor não se responsabiliza por:

- perdas financeiras;
- decisões tomadas com base nas informações geradas pelo aplicativo;
- danos decorrentes do uso do software.

---

## 7. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada a qualquer momento para refletir melhorias no software ou mudanças legais.

A versão mais recente estará sempre disponível na página oficial do aplicativo.
